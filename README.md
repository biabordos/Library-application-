A modern full-stack library management system designed to simplify book administration, borrowing workflows, and user management through an intuitive and scalable web application.
This project provides a complete environment for managing library resources, allowing administrators and users to interact efficiently with books, categories, authors, and borrowing operations.

Overview

The Library Management Application is built to streamline common library operations such as:
- Managing books and authors
- Tracking borrowed books
- Handling user interactions
- Organizing categories and inventory

The application follows a structured architecture that separates frontend, backend, and database responsibilities, making the project easier to maintain and extend.

Features

User Features
- Browse available books
- Search books by title, author, or category
- View detailed book information
- Borrow and return books
- Track borrowed books history

Admin Features
- Add, update, and delete books
- Manage authors and categories
- Manage library users
- Monitor borrowed and returned books
- Maintain inventory availability

Authentication & Security
- User authentication system
- Role-based access handling
- Secure database interactions
